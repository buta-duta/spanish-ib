-----------------------------------
FEATURE 35: IB LISTENING PRACTICE MODE
-----------------------------------

GOAL:
Simulate IB Spanish B listening exam practice using AI-generated or user-provided audio passages.

-----------------------------------
PASSAGE INPUT OPTIONS
-----------------------------------

Provide TWO options:

1. USER INPUT
- User can paste a Spanish passage (max ~400 words)

2. AI GENERATION
- Button: "🎧 Generate Listening Passage"
- AI generates a passage based on one of the 5 IB themes:
  - Identidades
  - Experiencias
  - Ingenio humano
  - Organización social
  - Compartir el planeta

- Passage requirements:
  - Natural spoken-style Spanish
  - Clear context (interview, news, conversation, announcement)
  - Length under 400 words

-----------------------------------
AUDIO PLAYBACK SYSTEM
-----------------------------------

- Convert passage into audio using text-to-speech

-----------------------------------

USER CONTROLS:
- "▶️ Play Audio"
- "⏸ Pause Audio"

-----------------------------------

SPEED CONTROL:
- Allow user to select playback speed:

Options:
0.8x, 1x, 1.25x, 1.5x, 2x, 3x, 4x

- Default = 1x

-----------------------------------

IMPORTANT:
- Audio ONLY plays when user clicks "Play"
- Do NOT autoplay

-----------------------------------
