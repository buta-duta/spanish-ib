FEATURE 12: VOICE-ONLY INPUT MODE
-----------------------------------

- The app must ONLY accept user input via voice (speech-to-text)
- Disable or remove text input fields entirely
- User interaction flow:
  1. User taps "Start Speaking"
  2. Speech is transcribed in real time
  3. Transcript is displayed on screen
  4. Submission happens automatically when user presses "Stop"

- All AI responses must:
  - Be displayed as text
  - Be spoken aloud in Spanish (text-to-speech)

-----------------------------------
FEATURE 13: PRACTICE PROGRESS BAR
-----------------------------------

Add a progress bar fixed at the top of the screen during practice.

The progress bar must show:
- Current progress through the session (visual bar)
- Number of questions remaining
- Estimated time remaining in MINUTES (not a countdown)

IMPORTANT:
- Do NOT display a live countdown timer
- Instead show static estimates, for example:
  "3 preguntas restantes • ~4 minutos restantes"

- Update this after each completed question

-----------------------------------
FEATURE 14: REGENERATE OPTIONS
-----------------------------------

Add two regeneration controls:

1. REGENERATE QUESTION
- Button visible after each AI question
- Replaces the current question with a new one from the same theme
- Must not repeat previous questions

2. REGENERATE SESSION (MAIN IDEA)
- Button available at any time during session
- Resets the conversation
- Generates:
  - New image
  - New theme (if random mode)
  - New set of questions

-----------------------------------
FEATURE 15: END-OF-SESSION FEEDBACK
-----------------------------------

After the practice session ends, AI must provide structured feedback in English.

Feedback must include:

1. LANGUAGE ANALYSIS
- Grammar mistakes (specific corrections)
- Tense usage (what was used vs what should be used)
- Vocabulary range (basic vs advanced)

2. IMPROVEMENT SUGGESTIONS
- Better sentence structures
- More advanced connectors
- Suggested higher-level vocabulary

3. IB CRITERIA ESTIMATION
- Criterion A: Language
- Criterion B: Message
- Criterion C: Conceptual Understanding
- Criterion D: Interaction

4. EXAMPLES
- Show 2–3 improved versions of user sentences in Spanish

-----------------------------------
FEATURE 16: EXIT CHAT OPTION
-----------------------------------

- Add a clearly visible "Exit Session" button

When clicked:
- Immediately stop current session
- Save progress (optional)
- Return user to main menu

- Include confirmation prompt:
  "¿Estás seguro de que quieres salir?"

-----------------------------------
GOAL:
Ensure a fully voice-driven, exam-realistic experience with:
- Clear progress tracking
- Flexible practice (regeneration options)
- Actionable IB-level feedback
- Full user control (exit anytime)
